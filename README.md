# Voice Based Concept Understanding Analyser

## Project Description
This application evaluates a user's spoken explanation using AI. It converts speech to text, analyzes conceptual understanding, calculates performance metrics, and generates reports and certificates.

## Features
- Speech-to-Text using Whisper
- Concept Understanding Analysis
- Performance Dashboard
- Speech Statistics
- PDF Report Generation
- Performance Certificate

## Technologies Used
- Python
- Streamlit
- Whisper AI
- Sentence Transformers
- Librosa
- ReportLab

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
